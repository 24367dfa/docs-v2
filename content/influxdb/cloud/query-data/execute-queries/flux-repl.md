The [Flux REPL](/influxdb/cloud/tools/repl/) starts an interactive
Read-Eval-Print Loop (REPL) where you can write and execute Flux queries.

```sh
./flux repl
```

For more information, see [Use the Flux REPL](/influxdb/cloud/tools/repl/).
