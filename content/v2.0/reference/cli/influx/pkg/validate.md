The `influx pkg validate` command validates the provided package.

## Usage
```
influx pkg validate [flags]
```

## Flags

| Flag              | Description                                 | Input Type |
|:----              |:-----------                                 |:---------- |
| `-f`, `--file`    | Package file to validate                   | string     |
| `-h`, `--help`    | Help for the `validate` command                  |            |
