### Update a variable

1. Click the **Organizations** icon in the navigation bar.

  {{< nav-icon "orgs" >}}

2. Select an organization from the list.  
3. Select the **Variables** tab.
4. Click on a variable's name from the list.
5. Update the variable's name and query.
6. Click **Submit**.
