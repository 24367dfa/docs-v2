Delete an existing variable in the InfluxDB user interface (UI).

### Delete a variable

1. Click the **Settings** icon in the navigation bar.

    {{< nav-icon "settings" >}}

2. Select the **Variables** tab.
3. Hover over a variable and click the trash can icon.
